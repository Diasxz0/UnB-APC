# APC - Algoritmos e Programação de Computadores (UnB)

Este repositório contém meus exercícios e projetos desenvolvidos durante a disciplina de APC na Universidade de Brasília (FCTE). 

## 🚀 Tecnologias
* Python 3.x
* Manipulação de arquivos (Built-in I/O)

## 📂 Organização
* **Logica e Condicionais:** Exercícios iniciais focados em `if/else`.
* **Repetição:** Uso de laços `for` e `while` para processamento de listas.
* **Arquivos:** Scripts que demonstram como ler (`r`), escrever (`w`) e anexar (`a`) dados em arquivos externos.
