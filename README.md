# 🏛️ APC - Algoritmos e Programação de Computadores (UnB)

Este repositório centraliza meus estudos, exercícios e projetos desenvolvidos na disciplina de **APC** na Universidade de Brasília (FCTE). O foco aqui é a base sólida em lógica de programação utilizando **Python**.

## 🚀 Tecnologias e Bibliotecas
* **Linguagem:** Python 3.10+
* **Matemática/Dados:** NumPy, Math, Random
* **Sistema/Automação:** OS, Shutil, Getpass
* **Interface Gráfica:** Pillow (PIL)

## 📂 Estrutura do Repositório
O repositório está organizado por módulos temáticos para facilitar a consulta:

* `logica-basica-e-condicionais`: Fundamentos e estruturas de decisão.
* `estruturas-de-repeticao`: Processamento de listas e loops (`for`/`while`).
* `manipulacao-de-arquivos`: Leitura, escrita e persistência de dados em `.txt`.
* `modulos-e-bibliotecas`: Funções utilitárias e cálculos com NumPy.
* `simulacoes-e-estatistica`: Lógica matemática e geração de dados aleatórios.
* `automacao-e-sistema`: Scripts para produtividade e gestão de arquivos.
* `bibliotecas-graficas`: Geração e manipulação de imagens via código.
* `jogos-e-desafios`: Implementação de jogos clássicos e desafios de lógica.
* `projetos-completos`: Sistemas mais robustos que integram múltiplos conceitos.
